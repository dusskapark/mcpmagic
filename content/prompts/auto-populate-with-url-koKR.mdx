# Prompt

- @https://www.grab.com/sg/에서 푸터의 사이트 맵 찾아줘
- 그리고 거기서 모든 텍스트 추출하고
- 선택한 노드에서 모든 텍스트 스캔한 다음에
- 그리고 스캔한 모든 텍스트를 추출한 푸터의 텍스트로 한 번에 업데이트해줘.

# How to Use

1. URL을 준비하세요.
2. 채울 노드를 선택하세요.
3. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
4. 실행 중..
5. 결과 확인

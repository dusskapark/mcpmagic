# Prompt

- 피그마에 접속해서 현재 선택 항목 가져와
- `Nav item 1`을 소스 인스턴스로 사용해줘.
- 다른 슬롯 블록을 소스 인스턴스로 오버라이드 해줘
- 추가로 붙여넣은 블록의 텍스트들을 이 디자인의 맥락에 맞춰서 다시 작성해줘

# How to Use

1. Prepare a source and target instances
2. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
3. 실행 중...
4. 결과 확인

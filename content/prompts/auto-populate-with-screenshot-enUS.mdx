# Prompt

- Understand the attached image and extract the text.
- Read the design of the currently selected node, scan the text, and appropriately replace it with the text extracted from the image.
- For meaningless nodes that don't need replacement, delete them.

# How to Use

1. Screen capture image 
2. Select nodes to fill 
3. Run the prompt: Copy and paste this prompt into the MCP interface 
4. Running 
5. Result

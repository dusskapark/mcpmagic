# Prompt

- Talk to figma, get the currently selected node
- Read and understand this design
- Scan all the text from the selected nodes.
- Parse the data from @path-your-file and use the data to replace multiple texts in Figma in bulk.


# How to Use

![give-me-figjam-connectors](/assets/give-me-figjam-connector.png)

1. Prepare a prototype noodles
2. Prepare FigJam Connector
3. Select nodes to fill
4. Run the prompt: Copy and paste this prompt into the MCP interface
5. Running
6. Result

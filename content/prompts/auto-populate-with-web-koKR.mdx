# Prompt

- 채널에 연결하고 선택한 노드를 찾습니다.
- 선택한 모든 텍스트를 스캔합니다.
- @Web에서 Spotify의 앨범 정보를 검색하고 BlackPink의 Rose의 앨범 APT를 찾습니다.
- 검색한 Spotify의 앨범 정보를 사용하여 여러 텍스트를 한번에 교체합니다.

3. 실행 중...

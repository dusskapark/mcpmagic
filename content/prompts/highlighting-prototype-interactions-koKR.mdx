# Prompt

- 피그마에 접속해서 현재 선택을 확인합니다. 
- 그것들의 프로토타입 리액션을 가져오고, 선택들 간의 연결관계를 이해합니다. 
- 그 노드들의 인터렉션을 설명하기 위한 주석을 설정해줘. 그리고 그것들의 대략적인 카테고리도 알아서 넣어줘.


# How to Use

> 🚨 Attention please
> 주석 기능은 유료 플랜인 프로 이상에서만 제공됩니다.
> ![Attention](/assets/attention-please.png)



1. 프로토타입을 준비하고 주석을 달 노드를 선택하세요
2. Run the prompt: Copy and paste this prompt into the MCP interface
3. Running
4. Result

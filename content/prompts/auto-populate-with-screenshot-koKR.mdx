# Prompt

첨부한 이미지를 이해하고 텍스트를 추출해줘.
현재 선택한 노드의 디자인을 읽고, 텍스트를 스캔한 다음 추출한 텍스트로 알맞게 대체해줘.
교체가 필요 없는 의미없는 노드들은 삭제해줘. 만약 삭제가 불가능하면 주석을 남겨줘.

# How to Use


1. 이미지를 캡쳐하세요.
2. 채울 노드를 선택하세요.
3. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
4. 실행 중..
5. 결과 확인

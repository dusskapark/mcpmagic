# Prompt

- 피그마에 접속해서 현재 선택된 노드들을 가져와
- 선택 항목에서 프로토타입 리엑션을 찾고 분석해줘
- 피그마 프로토타입 연결을 FigJam 커넥터로 변환해줘


# How to Use

![give-me-figjam-connectors](/assets/give-me-figjam-connector.png)

1. 프로토타입을 미리 연결해서 준비해주세요.
2. FigJam 커넥터를 준비하세요
  * 이 커넥터 기능을 사용하려면 FigJam 커넥터를 페이지에 복사하여 붙여넣어야 합니다. 한번만 설정이 완료되면 향후 사용에 문제가 없습니다.
3. Select nodes to fill
4. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
5. 실행 중..
6. 결과 확인

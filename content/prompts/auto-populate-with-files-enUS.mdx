# Prompt

- Talk to figma, get the currently selected node
- Read and understand this design
- Scan all the text from the selected nodes.
- Parse the data from @path-your-file and use the data to replace multiple texts in Figma in bulk.


# How to Use

1. Prepare a data file 
2. Select nodes to fill
3. Run the prompt: Copy and paste this prompt into the MCP interface
4. Running
5. Result

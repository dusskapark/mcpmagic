# Prompt

피그마에 접속해서, 선택한 노드에 대한 간단한 주석을 만들어줘. 엔지니어가 주석들을 통해서 디자인의 맥락을 이해할 수 있도록 도와줘.


# How to Use


> 🚨 Attention please
> 주석 기능은 유료 플랜인 프로 이상에서만 제공됩니다.
> ![Attention](/assets/attention-please.png)



1. 주석을 추가할 노드를 선택하세요
2. 이 프롬프트를 복사하고 MCP 인터페이스에 붙여넣기 해서 실행하세요.
3. 실행 중...
4. 결과 확인
